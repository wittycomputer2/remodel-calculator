# Contributing to Remodel Calculator

Thank you for your interest in contributing to this open-source remodel material calculator!

## How to Contribute
1. Fork the repository.
2. Create a new branch for your changes (`git checkout -b feature/your-feature`).
3. Make your improvements (e.g., bug fixes, new features, or documentation updates).
4. Commit your changes (`git commit -m "Add your commit message"`).
5. Push to your branch (`git push origin feature/your-feature`).
6. Open a Pull Request describing your changes.

We appreciate contributions that improve calculations, add new material types, or enhance the UI. Please ensure your code follows best practices and includes tests if applicable.
